# 快速安装指南

## 1. 加载扩展

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 打开右上角的"开发者模式"开关
4. 点击"加载已解压的扩展程序"
5. 选择 `/Desktop/simple-sync-clicker` 文件夹
6. 扩展加载成功！

## 2. 使用方法

### 第一步：打开两个交易平台

- Lighter: https://app.lighter.xyz
- Variational: https://omni.variational.io

### 第二步：手动设置交易参数

在**两个平台**上分别：
1. 选择交易对（如 ETH-PERP）
2. 选择方向（买入/卖出，做多/做空）
3. 输入数量
4. **不要点击提交！**

### 第三步：选择提交按钮（首次使用）

#### 在 Lighter 页面：
1. 点击页面右上角的**紫色悬浮按钮**
2. 页面进入选择模式（鼠标悬停会高亮元素）
3. **点击提交按钮**（通常是"确认"或"下单"按钮）
4. 看到提示："✅ Lighter 提交按钮已保存！"

#### 在 Variational 页面：
1. 点击页面右上角的**粉色悬浮按钮**
2. 页面进入选择模式
3. **点击提交按钮**（通常是"买 ETH"或"卖 ETH"按钮）
4. 看到提示："✅ Variational 提交按钮已保存！"

### 第四步：执行同步点击

1. 点击 Chrome 扩展图标（工具栏）
2. 弹出控制面板
3. 确认状态：
   - Lighter: ✅ 已就绪
   - Variational: ✅ 已就绪
4. 点击"⚡ 执行同步点击"按钮
5. 两个平台会**在同一时刻点击提交按钮**！

## 3. 重要提示

### ✅ 优势
- 不依赖 DOM 结构，平台更新不影响使用
- 用户完全掌控交易参数
- 简单可靠，只负责同步点击

### ⚠️ 注意事项
- 每次交易前需要**手动设置好参数**
- 首次使用需要选择一次提交按钮
- 如果页面刷新，需要**重新选择按钮**
- 建议先用**小额测试**（0.001-0.01 ETH）

### 🔄 重置按钮
如果需要重新选择按钮：
1. 打开扩展弹窗
2. 点击"🔄 重新选择按钮"
3. 页面会刷新
4. 重复"第三步"重新选择

## 4. 同步精度

- **缓冲时间**: 300ms
- **目标精度**: < 5ms
- **实际精度**: 取决于网络和系统性能

## 5. 故障排查

### 问题：看不到悬浮按钮
- **解决**：刷新页面，确保扩展已加载

### 问题：扩展弹窗显示"未就绪"
- **解决**：检查是否已在两个页面都选择了提交按钮

### 问题：点击后没有反应
- **解决**：
  1. 检查是否已手动设置好交易参数
  2. 确保选择的是提交按钮（不是其他按钮）
  3. 尝试重新选择按钮

## 6. 开发者调试

打开 Chrome 开发者工具（F12），查看 Console 日志：

- `[Lighter Sync]` - Lighter 页面日志
- `[Variational Sync]` - Variational 页面日志
- `[Background]` - 后台服务日志
- `[Popup]` - 弹窗日志
